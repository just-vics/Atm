A fully functional ATM simulation built in C# (.NET 9) that allows users to create accounts, log in securely, check balances, deposit, withdraw, and transfer money between accounts or to other users. The system includes unique account and card number generation, input validation with custom exceptions, and a hidden admin mode (PIN: 2004) for viewing and managing all users. With loading animations, clean console UI, and 5-attempt PIN lockout, this project demonstrates OOP principles, error handling, and real-world banking logic in a simple console environment. Perfect for learning or showcasing C# fundamentals.
